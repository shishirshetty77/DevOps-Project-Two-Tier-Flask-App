About This Project

This is a two-tier web application built with a Flask backend and a PostgreSQL database. The entire application is containerized using Docker and orchestrated with Docker Compose. The backend is served by a production-ready Gunicorn WSGI server, and database connections are managed efficiently using a connection pool.
